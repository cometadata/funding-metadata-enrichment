# Funding Metadata Enrichment

An open pipeline for producing structured funding metadata from the full text. Given a corpus of works, locates funding statements and converts them into structured funding metadata. 

Part of the COMET initiative pilot project: 'Improve Funding Metadata'. Refer to the [Project Hub](https://docs.google.com/document/d/1Secq8-wVkPcy8aY1kYY_OzdAYleUKY6yDqzG-9lryrw/edit?usp=drive_link)
